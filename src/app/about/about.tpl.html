What is it?

This app is meant to be built during the San Diego JS Ride Along scheduled for 02/09/2015.

To run this app there is only one requirement.

  • Vagrant

-- OR --

If you don't want to download Vagrant and run the included Vagrantfile, just make sure you have the following installed globally.

  • Grunt & Grunt CLI
  • Karma
  • Bower
  • Karma compatible browser
